Harvest Clicker

Welcome! To launch and play Harvest Clicker, your local development server needs to be running.

Open Game (http://localhost:9002)
If the game doesn't open:
  1. Open your project terminal.
  2. Run the command: npm run dev
  3. Once the server starts (usually indicated by a message like "ready - started server on 0.0.0.0:9002, url: http://localhost:9002"), click the button above or open http://localhost:9002 in your browser.
